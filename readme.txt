log格式规范：

1.最好是英文字符
2.一次发送的log要写成一行的形式，最后要加上换行\n
3.变量名尽量有区分度，不要多个变量名一样


1.标准格式1：变量名1：数值1，变量名2：数值2，...                                     （【变量+冒号+数值+逗号/分号】为一个键值对）

2.标准格式2：变量名1：数值1，变量名2：（数值01，数值02，数值03）,....      （即变量对应的数值可以是多个值，用括号和逗号分开）

3.标准格式3:变量名1：数值1 单位1，变量名2：数值2 （单位2），...                   (即单位可以直接写如:cm、db、m，或：（cm）、（db）、（m）) 

例：
1.
	Idx:130439,Dis:257.92cm,MPF:16,INIT:Azi:45.23deg,PDOA:(-31.06)deg,RSSI:(-78,-74)dBm,Gain_idx:0
2.
	Idx:21380,D:119.99cm,MPF:0,INIT:Azi:-19.02deg,Ele:17.74deg,PDOA:(-78.20,-25.06,53.79)deg,RSSI:(-66,-66,-66)dBm,Gain_idx:1,Temp:36.3C;RESP:Azi:9.84deg,Ele:-0.74deg,PDOA:(-55.72,-21.40,34.18)deg



多个数值只能用（x1,x2,x3）表示，如果有单位只接受直接写或放括号中： 单位或(单位)，不接受数值、单位，变量放在”“、‘  ’、* *、等其他符号中
